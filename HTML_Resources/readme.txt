Пример разработки инструмента для создания и редактирования DLL ресурсов для хранения в них файлов HTML / CSS / JS
и изображений (JPEG, GIF, PNG, ...). DLL могут быть использованы в любой программе или с TWebBrowser / TEmbeddedWB.

Компоненты, необходимые для компиляции программы:
- TembeddedWB: http://sourceforge.net/projects/embeddedwb/
- SpTbxLib: http://www.silverpointdevelopment.com/
- VitualTreeView: http://www.soft-gems.net/
- SynEdit: http://sourceforge.net/projects/synedit/
